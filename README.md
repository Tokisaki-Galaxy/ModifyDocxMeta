## 前言

docx文件里面可以记录你写这个文档的编辑时间，修改人，修改时间。这个信息有时候会带来大问题。比如你用ai写了一篇2000字论文 但是老师一看总编辑时间只有10分钟，那你可能得好好解释咋写的了。
